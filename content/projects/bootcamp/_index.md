## Virginia Tech × Fullstack Academy

I'm currently enrolled in the [Virginia Tech Data Analytics Bootcamp](https://bootcamp.cpe.vt.edu/programs/data-analytics) — a rigorous AI-powered program delivered in partnership with Fullstack Academy, recognized as one of Course Report's 2025 Best Online Bootcamps.

The program spans the full modern analytics stack: from business analytics in Excel to SQL for data engineering, Python and R for analysis and machine learning, and Tableau and Power BI for visualization.

## What I'm learning

The curriculum is hands-on — every unit ends with a career simulation project graded by industry professionals.

### Excel & Business Analytics
Data cleaning, PivotTables, dashboarding, macros, Power Query, what-if analysis, and statistical functions. Culminating in a full e-commerce dashboard capstone.

### SQL for Data Acquisition
From fundamentals to advanced: joins, subqueries, Views, CTEs, and window functions. Simulated employee performance analysis project.

### Python Programming + Data Analytics
Core Python, then deep dives into NumPy, Pandas, statistics, and data visualization for real-world datasets.

### R for Statistical Analysis & Machine Learning
tidyverse, hypothesis testing, regression (simple/multiple/non-linear), classification (decision trees, SVM, Naive Bayes), clustering (K-means, hierarchical, DBSCAN), and association mining.

### Tableau — Prep for Tableau Desktop Specialist
LOD expressions, table calculations, advanced charts, dashboards, and data storytelling.

### Power BI (Elective)
Data modeling, DAX, row-level security, and advanced analytics — including Q&A, key influencers, and decomposition tree visuals.
